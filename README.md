  ## Usage
  1. Start the server by running node server.js. It will then tell you that it is running on port 3000. 
  2. Go to localhost/3000
  3. Type in a burger you would like to add. 
  4. This burger will then get added to the db and shown in the burgers to be devoured section.
  5. You can devour the burger and the state will be updated and it will move to devoured burgers.
